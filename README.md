# LIBRARIUM

Librarium es una plataforma de gestión y visualización de bibliotecas digitales desarrollada en Python. Este proyecto nace de un flujo de trabajo híbrido, combinando la eficiencia de la Inteligencia Artificial con una supervisión técnica rigurosa y diseño artesanal.

## Propósito del Proyecto

El objetivo de Librarium es ofrecer una interfaz limpia y funcional para la organización de catálogos digitales. Este repositorio es una prueba de concepto de cómo un desarrollador puede utilizar herramientas de IA para potenciar la lógica de programación sin sacrificar el control sobre la arquitectura y la estética del producto final.

## Mi Labor de Desarrollo

En este proyecto, he diferenciado claramente las responsabilidades para garantizar un código sólido y un diseño único:

* **Arquitectura de Datos (JSON):** Diseñé íntegramente la estructura del archivo JSON, asegurando una jerarquía lógica que permita una gestión de datos escalable y fácil de mantener.
* **Lógica de Backend (.py):** Utilicé la IA como un "copiloto" para profundizar en conceptos de Python y acelerar la escritura de funciones complejas. Mi labor consistió en la investigación, auditoría del código generado, depuración y la integración final de los módulos.
* **Diseño Frontend (CSS):** El apartado visual es **100% autoría propia**. He desarrollado el código CSS desde cero para lograr una interfaz moderna, profesional y responsive, alejándome de plantillas genéricas.

## Notas Técnicas y Transparencia

Como parte de un desarrollo honesto y profesional, cabe destacar que:
* **Gestión de Portadas:** La aplicación utiliza servicios externos para la recuperación de imágenes. Debido a la naturaleza de estas APIs, el sistema puede presentar omisiones o mostrar portadas incorrectas en casos específicos. He mantenido este comportamiento para demostrar mi capacidad de análisis sobre las limitaciones de servicios de terceros y la gestión de errores en entornos reales.

## Características Principales

* Visualización dinámica de catálogos basada en JSON.
* Interfaz de usuario personalizada con enfoque en la legibilidad.
* Estructura modular en Python para facilitar futuras expansiones.
* Diseño adaptable (Responsive Design).
